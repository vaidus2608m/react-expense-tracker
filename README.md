# Expense Tracker

A modern, minimal React-based Expense Tracker application designed with a professional SaaS dashboard aesthetic.

## 🔗 Live Demo
https://expenly-react.netlify.app/

## ✨ Features
- **Clean SaaS UI:** Professional, centered layout with card-style expense items.
- **Add & Manage Expenses:** Easily track expenses with title, amount, category, and date.
- **Categorization:** Filter or organize expenses by category.
- **Total Calculation:** Dynamically calculates and visually highlights total parsed expenses.
- **Responsive Design:** Fully responsive layout with polished typography, subtle shadows, and hover effects.
- **No External UI Libraries:** Custom styling using pure CSS for complete design control.

## 🛠️ Technologies Used
- **Frontend:** React, Vite
- **Styling:** Custom CSS (No external libraries)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd expense-tracker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to the local server URL provided in your terminal (usually `http://localhost:5173`).
